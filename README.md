# Qlik MCP Showcase

A demonstration and documentation site for Qlik's Model Context Protocol (MCP) integrations, built by Jandré Killian (Principal Consultant, Qlik Partner Ambassador).

The site shows how Claude AI + Qlik MCP can query live Qlik applications and generate real business intelligence outputs — dashboards, presentations, reports, and workbooks — from a single prompt.

---

## What's on the site

### Landing page (`index.html`)
The main entry point. Contains:
- **Demo videos** — screen recordings of MCP builds in action
- **Prompt library cards** — links to the two interactive prompt pages
- **Examples teaser** — preview grid of real outputs
- **Documentation** — downloadable Qlik MCP Capabilities PDF

### Prompt libraries
Two interactive pages, each with copyable prompts:

| Page | Theme | Prompts | Focus |
|------|-------|---------|-------|
| `qlik-mcp-power-prompts_1.html` | Light/editorial | 9 + 1 mega prompt | Advanced analytical patterns (anomaly detection, forecasting, risk scoring, scenario modelling, documentation) |
| `qlik-mcp-output-prompts_1.html` | Dark/tech | 14 + 1 hero prompt | External deliverables — HTML dashboards, PowerPoint decks, Word reports, Excel workbooks, email packs |

Prompts are filterable by format and include a copy-to-clipboard button, the Qlik MCP tools used, and an explanation of why the prompt works.

### Actual examples (`actual-examples.html`)
A tabbed gallery of real outputs produced by Qlik MCP using demo data:

- **HTML tab** — three interactive dashboards (defects, products, vendor network)
- **PowerPoints tab** — executive board pack
- **Word tab** — two documentation outputs (PDF preview)
- Excel, Mashups, Other tabs — coming soon

### Example dashboards (standalone HTML files)
All generated by Qlik MCP from a SAP procurement dataset (demo data):

| File | Type | Description |
|------|------|-------------|
| `defects-dashboard.html` | Chart.js dashboard | SAP defect intelligence — KPIs, trends, vendor/product breakdown, forecast |
| `defective-products-dashboard.html` | Chart.js dashboard | Defective products KPI view with 12-month linear regression forecast |
| `vendor-defect-network.html` | D3.js network graph | Force-directed graph linking 8 vendors to 28 products by defect volume and risk level |

## How to use it

1. Open `index.html` in a browser — no build step, no dependencies, pure HTML/CSS/JS
2. Watch the demo videos to see MCP in action
3. Browse the prompt libraries and click **Copy** on any prompt
4. Paste the prompt into Claude (with Qlik MCP connected) and run it against your Qlik app
5. See the Actual Examples page to understand the kind of output you can expect

---

## Tech stack

- Plain HTML, CSS, JavaScript — no framework, no bundler
- [Chart.js](https://www.chartjs.org/) for standard charts
- [D3.js](https://d3js.org/) for the network visualisation
